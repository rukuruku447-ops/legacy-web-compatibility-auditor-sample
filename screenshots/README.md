# Screenshots

ここに製品の実行画面スクリーンショットを追加してください。

## 推奨キャプチャ内容

- `scan_terminal.png` — `py -m ie_auditor scan` の実行結果（ターミナル画面）
- `audit_html.png` — `sample_report.html` をブラウザで開いた画面
- `self_test.png` — `py -m ie_auditor self-test` の Result: OK 画面

## キャプチャ方法（Windows）

1. コマンドプロンプト / PowerShell でコマンドを実行
2. `Win + Shift + S` でスクリーンショット
3. PNG として保存してこのフォルダに配置
